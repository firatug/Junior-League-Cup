# Junior League Cup

7 yaş futbol takımları için turnuva organizasyonu platformu.

**GitHub:** https://github.com/firatug/Junior-League-Cup

## Çözüm Yapısı

| Proje | Açıklama |
|-------|----------|
| **Junior League Cup Web Site** | ASP.NET Core MVC web sitesi (`src/JuniorLeagueCup.WebSite`) |

Gelecekte eklenecek: SQL veritabanı projesi ve diğer servisler.

## Gereksinimler

- [.NET 10 SDK](https://dotnet.microsoft.com/download) (veya uyumlu .NET SDK)

## Çalıştırma

```bash
cd src/JuniorLeagueCup.WebSite
dotnet run
```

Tarayıcıda `https://localhost:7xxx` adresini açın (port `launchSettings.json` içinde tanımlıdır).
